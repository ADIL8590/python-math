Probability Mass Function

A probability mass function (PMF) is a mathematical function that describes the probability distribution of a discrete random variable. It assigns probabilities to each possible value of the random variable, indicating the likelihood of each outcome occurring. The PMF is defined for discrete random variables and is used to calculate probabilities for specific values.

Properties of Probability Mass Functions

  • The PMF is non-negative for all values of the random variable.
  • The sum of the probabilities assigned by the PMF over all possible values of the random variable is equal to 1, which represents the total probability.
  • The probability that a random variable takes on a specific value can be directly obtained from the PMF.

Example: Binomial Distribution

One common example of a probability mass function is the binomial distribution. It models the number of successes in a fixed number of independent Bernoulli trials, where each trial has two possible outcomes: success or failure. The PMF of a binomial distribution is given by:

        P(X = k) = (n choose k) * p^k * (1 - p)^(n - k)
                 OR
        P(X = k) = (n! / (k! * (n - k)!)) * p^k * (1 - p)^(n - k)
        Here, (n choose k) is the binomial coefficient, which can be calculated as n! / (k! * (n - k)!), where n is the number of trials and k is the number of successes. The parameter p represents the probability of success on each trial. The binomial distribution is widely used in statistics and various fields to model scenarios involving binary outcomes.
  • Let's break down the formula for the PMF of a binomial distribution:
  • n: The total number of independent trials.
  • k: The number of successes we want to calculate the probability for.
  • p: The probability of success on each trial.
  • (n choose k): The binomial coefficient, which represents the number of ways to choose k successes from n trials. It can be calculated using the formula n! / (k! * (n - k)!).
  • p^k: The probability of having k successes.
  • (1 - p)^(n - k): The probability of having (n - k) failures, which is the complement of success.

First Moment (Mean)

  1. The first moment of a probability distribution is the expected value or mean of the distribution. It provides a measure of the central tendency of the distribution.
  2. In the case of a binomial distribution, the first moment (mean) is given by n * p, where n is the number of trials and p is the probability of success on each trial.

Second Moment (Variance)

  1. The second moment of a probability distribution is the variance, which measures the spread or variability of the distribution around its mean.
  2. For a binomial distribution, the second moment (variance) is given by n * p * (1 - p), where n is the number of trials and p is the probability of success on each trial.
